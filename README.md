# Stock Calc (Java, BlueJ)

A java project that simulates selling and buying shares of stock. This projects uses multiple different queue implementations and utilizes key object orientated concepts like encapsulation, poloymorphism and abstraction. This project was created in BlueJ and allows methods such as buying, selling, viewing the total realized capital gain, and quitting.
---
## Features

- Built a Java application that simulates a stock market that uses multiple queue implementations
- Applied an object oriented design to manage items throughout multiple different queue implementations
- Utilized queue operations to demonstrate a practical application of the data structure in Java
- The user will be prompted to enter how many shares they wish to buy and at what price.
- The user can also be able to enter how many shares they wish to sell and at what price.
---

## Technologies Used
- **Java**  
- **BlueJ**
